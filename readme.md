# ping pong
ping pong is a simple Pong clone I made in Python with the turtle wrapper for tkinker
## Usage
- Windows:
    - Download main.exe from releases and run it I guess?
- Other
    - Make sure tkinker is installed, if it is run main.py
## Features
- There's a main menu for stuff
- 2 player support (because there is no ai to play with you)
